Project 1 : Chain Game

Browser based game

For my first project, I had one week to devise a game concept then build to completion. This is a memory based game that utilised HTML5, CSS3, SASS, JavaScript and jQuery. The computer generates a random sequence to light up the cells in order, the user then has to repeat the same order to progress to the next level and increase the sequence length by one.

Project 4 : ABC Events

MERN stack app

My final project on the course was completed in a group of three, using React for the first time. This web app is designed to be a marketplace for buying, selling or looking at tickets for a wide range of global events. The app included API’s from Paypal for payment, MailChimp Subscription Service to send out newsletters and Google Map with Autocomplete for event location. The Ticketmaster and EventBrite API’s are also implemented in case the event that you are looking for is currently not available on the app. This was my first project as Project Manager, which I thoroughly enjoyed and look forward to taking up when given the opportunity.
